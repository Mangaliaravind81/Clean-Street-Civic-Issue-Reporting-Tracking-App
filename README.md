🚀 How to Run the Project

Follow the steps below to run the application locally.

📌 Step 1: Open Two Terminals

You need two separate terminals — one for the backend and one for the frontend.

🖥️ Backend Setup
1️⃣ Navigate to the Project Folder
cd Clean-Street-Civic-Issue-Reporting-Tracking-App-main
2️⃣ Go to the Backend Directory
cd backend
3️⃣ Install Nodemon Globally (if not installed)
npm install -g nodemon
4️⃣ Start the Backend Server
nodemon app.js

This will connect the backend to the MongoDB server.

🌐 Frontend Setup
1️⃣ Open the Second Terminal

Navigate to the project folder:

cd Clean-Street-Civic-Issue-Reporting-Tracking-App-main
2️⃣ Go to the Frontend Directory
cd frontend
3️⃣ Install Dependencies (if not installed)
npm install
4️⃣ Start the Frontend
npm run dev

The frontend will now run locally (usually at http://localhost:5173).
